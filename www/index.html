webserv

Serveur HTTP en C++98, non-bloquant, un seul epoll() pour toutes les I/O. Racine servie : www, port 8080.

Pour la démonstration complète (tests automatisés couvrant tout le sujet), aller sur /default/ — lancer le serveur avec ./webserv conf/demo.conf pour débloquer tous les scénarios (413, 403, redirection, second port, etc.).
